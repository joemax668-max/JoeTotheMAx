Troubleshooting

  • Make sure your computer is online — these images are hotlinked from Wikimedia Commons and the videos are YouTube embeds.
  • If images still don't appear, try opening the page from a simple local server (some browsers block embeds when opened as file://). Example: use the VS Code Live Server extension or run `python -m http.server` in the folder and open `http://localhost:8000`.
  • Some corporate or school networks block external sites — try a different network if nothing loads.
